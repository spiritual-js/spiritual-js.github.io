tick

The tick plugin deals with the timing of things. This plugin is in a sorry state, so let's just give a quick overview of some API calls that are not likely to change.

Ticks

this.tick.next(function() {
	this.dom.hide();
});

Timeouts

this.tick.time(function() {
	this.dom.hide();
}, 23);

Animations

this.tick.nextFrame(function() {
	this.css.left += 5;
});
